# Wireservers UI

A React Native component library built with NativeWind and Tailwind Variants.

## Packages

| Package | Description |
| --- | --- |
| [`@wireservers-ui/react-natives`](packages/react-natives/) | Shared React Native component library |
| [`demos/expo`](demos/expo/) | Expo demo app |

## Getting Started

```bash
# Install dependencies
npm install

# Start the demo app
cd demos/expo
npx expo start
```
